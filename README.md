# Portfolio Personnel

Site web portfolio professionnel presentant mon parcours academique, mes competences techniques et mes projets.
https://ghofranetriki17.github.io/portfolio/

## Description

Portfolio interactif et responsive developpe dans le cadre de ma formation en Genie Informatique a l'IIT Sfax.

## Technologies utilisees

| Technologie | Utilisation |
|-------------|-------------|
| HTML5 | Structure semantique |
| CSS3 | Design responsive, animations, variables CSS |
| JavaScript | Interactivite et logique |
| Font Awesome | Icones |
| Google Fonts | Typographies |

## Fonctionnalites

- Design responsive (mobile, tablette, desktop)
- Animations CSS et transitions fluides
- Effet de frappe dynamique
- Systeme de modales interactives
- Navigation smooth scroll
- Menu mobile hamburger
- Bouton retour en haut de page
- Header fixe avec effet au scroll

portfolio/

## Installation

1. Cloner le repository
   ```bash
   git clone https://github.com/ghofranetriki17/portfolio.git
   ```

2. Acceder au repertoire
   ```bash
   cd portfolio
   ```

3. Ouvrir `index.html` dans un navigateur

## Deploiement

Le site peut etre deploye sur GitHub Pages :

1. Aller dans Settings du repository
2. Section Pages
3. Selectionner la branche main comme source
